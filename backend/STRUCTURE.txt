Key Features:
============
✓ Async/await throughout (AsyncSession, async def)
✓ Type-safe (mypy compatible)
✓ CORS configured for frontend
✓ Error handlers (HTTP & general exceptions)
✓ Environment-based configuration
✓ Health check endpoint
✓ Proper relationships & cascade deletes
✓ JSON column support for metadata
✓ Timezone-aware timestamps
✓ Session lifecycle management
✓ Pydantic validation on all inputs
✓ OpenAPI/Swagger docs auto-generated

Database Tables:
===============
1. websets           - Webset collections
2. webset_items      - Items in websets
3. monitors          - Scheduled monitors
4. monitor_runs      - Monitor execution history
5. extraction_jobs   - Extraction task tracking

All tables have proper:
- Primary keys (TEXT)
- Foreign keys with CASCADE
- JSON columns for metadata
- Timestamps (created_at, updated_at)
- Indexes (automatic via relationships)
